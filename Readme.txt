# Gourmet_Atlas

## Description du projet

Gourmet_Atlas est une application qui permet aux utilisateurs d'explorer la cuisine mondiale. En utilisant l'API World Food Guide, notre application offre des informations détaillées sur une variété de cuisines et de plats du monde entier.

## Fonctionnalités

- **Explorer les cuisines** : Les utilisateurs peuvent explorer différentes cuisines du monde entier, y compris des informations détaillées sur chaque cuisine.
- **Découvrir les plats** : Les utilisateurs peuvent découvrir une variété de plats, y compris leurs ingrédients et comment ils sont préparés.
- **Galerie de photos** : Les utilisateurs peuvent voir des photos de différents plats pour avoir une idée de leur apparence.

## Comment lancer le projet

1. Clonez le dépôt : `git clone https://github.com/Boushow/Gourmet_Atlas-Project.git`
2. Accédez au répertoire du projet : `cd Gourmet_Atlas-Project`
3. Installez les dépendances : `npm install`
4. Lancez le serveur : `npm start`
5. Ouvrez votre navigateur et accédez à `http://localhost:3000`
